🌊 AquaFun WaterPark Booking Fullstack Website

A full-stack Water Park Booking Web Application developed using modern web technologies. This project allows users to explore water park details, book tickets, and manage reservations easily.

🚀 Features
🏖️ Attractive Home Page with water park theme
🔐 User Login & Registration System
🎟️ Online Ticket Booking
📅 Booking Management System
📡 REST API Integration
💾 Data stored using JSON (Mock Database)
⚡ Fast and responsive UI using React.js
